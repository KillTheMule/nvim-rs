Struct nvim_rs::compat::tokio::Compat

pub struct Compat<T> { /* private fields */ }
Available on crate feature use_tokio only.
Expand description

A compatibility layer that allows conversion between the tokio::io and futures-io AsyncRead and AsyncWrite traits.
